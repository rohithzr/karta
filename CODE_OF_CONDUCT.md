# Code of Conduct

This project follows the [Contributor Covenant, version 2.1][covenant].
Please read the full text before participating.

In short: be respectful, assume good intent, disagree about technical
decisions without making it personal, and keep the project welcoming
to newcomers. Maintainers will enforce these standards.

To report a concern, open a private issue or email the maintainer listed
in `Cargo.toml`.

[covenant]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
